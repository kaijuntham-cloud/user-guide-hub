To add or edit guides:

1. Right-click this page and select "View Page Source" or "Inspect"

2. Find the guides array in the JavaScript section

3. Add new guides or modify existing ones using this format:

{
    title: "Guide Title",
    description: "Brief description of what this guide covers",
    category: "Department/Category",
    pdfUrl: "path/to/your/file.pdf"
}

4. Save the file and refresh the page

Note: Replace the pdfUrl with either a relative path (if PDFs are in the same folder) or a full URL to where your PDF is hosted.
