&nbsp; &nbsp; &nbsp; &nbsp; &nbsp;  üìù How to Update Guides
&nbsp; &nbsp; &nbsp; &nbsp; &nbsp;  To add or edit guides:

&nbsp; &nbsp; &nbsp; &nbsp; &nbsp;  1. Right-click this page and select "View Page Source" or "Inspect"

&nbsp; &nbsp; &nbsp; &nbsp; &nbsp;  2. Find the guides array in the JavaScript section

&nbsp; &nbsp; &nbsp; &nbsp; &nbsp;  3. Add new guides or modify existing ones using this format:

&nbsp; &nbsp; &nbsp; &nbsp; &nbsp;  
{
&nbsp;  title: "Guide Title",
&nbsp;  description: "Brief description of what this guide covers",
&nbsp;  category: "Department/Category",
&nbsp;  pdfUrl: "path/to/your/file.pdf"
}
&nbsp; &nbsp; &nbsp; &nbsp; &nbsp;  4. Save the file and refresh the page

&nbsp; &nbsp; &nbsp; &nbsp; &nbsp;  Note: Replace the pdfUrl with either a relative path (if PDFs are in the same folder) or a full URL to where your PDF is hosted.
